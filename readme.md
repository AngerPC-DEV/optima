***----------------------------------------------------------------------***
1. РОЗПАКУВАННЯ ТА ПІДГОТОВКА
***----------------------------------------------------------------------***
Створюємо робочу директорію, переносимо туди архів, розпаковуємо його та видаляємо оригінальний .zip:

mkdir github
cd github
# Припускаємо, що архів github.zip знаходиться у батьківській папці
cp ../github.zip .
unzip github.zip
rm github.zip

Щоб подивитися список доступних проєктів, виконайте команду:
ls

Список проєктів у репозиторії:
* FLAPPYBIRD — відома пташка, що літає.
* DINO — класичний динозавр Google.
* ARKANOID — кулька, яку відбиває платформа.
* TELEGRAM — мій телеграм-бот, знятий з продакшену спеціально для презентації ☺️

***⚠️ ВАЖЛИВО: З міркувань безпеки та захисту приватної архітектури (private source), цей репозиторій містить лише демонстраційну базу (модулі ban, mute, kick, unban, unmute). Понад половину приватних модулів (система привітання, економіка видобутку COIN, віртуальний магазин ShopInBot, токени та шляхи до баз даних/сайту) було видалено задля уникнення витоку даних. Код авторизації у /start вимкнено.***

📡 [1] TELEGRAM BOT
----------------------------------
cd TELEGRAM
pip install aiogram asyncio

***⚠️ Якщо виникає помилка "hint: See PEP 668...":
pip install aiogram asyncio --break-system-packages***

Перевірка встановленої версії aiogram:
##pip show aiogram##

Запуск бота:
**python main.py**

⚠️ Щоб зупинити бота, натисніть CTRL + C або виконайте:
`***pkill -f main.py***`
